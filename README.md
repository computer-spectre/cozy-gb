## Getting started
### Dependencies
In order to compile, you will need :
* `glfw`
* `glew`
* `glm`
### Compiling
Just run the Makefile using :
```bash
$ make
```
### Playing
```bash
$ ./bin/cozy-gb path/to/the/rom
```
